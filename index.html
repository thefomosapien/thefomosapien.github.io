My Skills

By definition I'm a MEAN stack developer but I love to learn new frameworks,
languages and other technologies.

  • JavaScript

    Javascript was my "gateway drug" to all things programming. You can see a lot of the toy problems I've solved on freeCodeCamp.com.

  • Angular

    Corey's my name. Modularization is my game. Angular directives cut my coding time in half and make my code base clean AF.

  • MongoDB

    Oh backend, you scared me at first. Now I embrace you with open arms like a dame, welcoming her sailor home.

  • Node

    Server-side JavaScript!? Count me in! I truely was terrified at first. Now I can't stop bulding full-stack applications.

  • HTML

    I was introduced to the structure of sites about 3 years ago and it started a journey of learning and discovery.

  • CSS

    During my education at DevMountain, I would overhear a lot of my peers complain about CSS. I have nothing but love. My favorite part is design.

  • Jquery

    Jquery is like the album, Chronic. So good, a classic. However, To Pimp a Butterfly (Angular) came along with more raw talent and potential. The latter wouldn't exist without the former though.

  • d3.js

    I worked closely with my team to build some great responsive graphs and charts for our Billing Simplified web app. The key to that app was the visuals.

  • Gulp

    My first experience with a robust build tool. A very intuitive and programmer-friendly tool, especially using Node packages.

  • SASS

    CSS on steroids. Actually, more like a super ripped and lean version of CSS? Sass has become my new best friend. I used both Gulp and Sass for my Billing Simplified web app.
